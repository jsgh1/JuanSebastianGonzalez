# Prueba Técnica

Este repositorio reúne la documentación y los recursos desarrollados durante el trabajo realizado en clase.

## Descripción

El proyecto contiene los avances, decisiones y entregables relacionados con la prueba técnica. La información está organizada en repositorios separados para facilitar la consulta del material documental y de la base de datos.

## Repositorios del proyecto

### Documentación

Repositorio con la documentación general del proyecto:

[PruebaTecnica-Docs](https://github.com/jsgh1/PruebaTecnica-Docs)

### Base de datos

Repositorio con los scripts, estructura y recursos asociados a la base de datos:

[PruebaTecnica-DB](https://github.com/jsgh1/PruebaTecnica-DB)

## Propósito

El objetivo de estos repositorios es centralizar el trabajo realizado, mantener un registro organizado de los entregables y facilitar la revisión del desarrollo del proyecto.

## Contenido general

- Documentación del proyecto.
- Decisiones técnicas.
- Recursos relacionados con la base de datos.
- Material trabajado durante las sesiones de clase.
